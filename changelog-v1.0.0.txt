
DownloadMedia v1.0.0

Release highlights:
- Added separate installers for x64, x86, and ARM64.
- Added Google Drive based update discovery and install flow.
- Added automatic update checks on app launch and while the app is open.
- Added version-aware changelog loading.
- Improved direct play, dropdown consistency, and player reuse behavior.
- Kept the app locked to a black and white theme.
- Improved packaging and production stability.

Security and reliability:
- Strengthened update validation around versioned release artifacts.
- Reduced architecture mismatch risk by isolating installer paths per build target.
- Improved binary resolution for Windows architectures.

Notes:
- The updater expects the matching release zip in the architecture folder.
- The changelog filename must match the version suffix exactly for automatic display.
